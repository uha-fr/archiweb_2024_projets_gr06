addUserPlan()

addUserPlan

public addUserPlan(array<string|int, mixed> $recipesData, int $period, int $duration, string|null $plan_name) : bool

This function adds a new plan for the user based on the provided data. If a plan name is not provided, it defaults to "Default Plan for User" followed by the user's ID. The plan details are inserted into the plans table, and the user-plan relationship is stored in the user_plan table. Additionally, each recipe in the plan is inserted into the plan_recipes table.

Parameters
$recipesData : array<string|int, mixed>

An array containing information about the recipes in the plan

$period : int

The number of days of the plan (repeats through the duration)

$duration : int

The total number of days of the plan

$plan_name : string|null

The name of the plan (optional)

Return values
bool —

Returns true if the plan is successfully added, false otherwise
